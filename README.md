# Robot Operating System

## How Can I Run
## Prerequisites

* Use `Pyhton3` standart 
* Download `Ros Noetic`.
```
roslaunch turtlebot3_gazebo turtlebot3_word.launch
rosrun mapping_tb3_node.py
```
